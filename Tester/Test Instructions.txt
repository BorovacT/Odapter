Tester Instructions

The tests should be run in order below.

Initialization for 4.0 Tests
----------------------------
Run /bin/x64/Debug/Odapter.exe and generate using _Odpt-NoFilter.config.


C# 4.0 Standard Test
--------------------

a) Run /bin/x64/Debug/Odapter.exe and generate using _Odpt.config.

b) Comment out all Define directives in Tester.cs except:

ODPT_FILTER_PREFIX
MAPPING_FOR_TYPED_CURSOR
SEED_TABLES

c) Compile and run Tester project


C# 4.0 No Filter Test
---------------------

a) Run /bin/x64/Debug/Odapter.exe and generate using _Odpt-NoFilter.config.

b) Comment out all Define directives in Tester.cs except:

MAPPING_FOR_TYPED_CURSOR
SEED_TABLES

c) Compile and run Tester project


C# 4.0 Safe Type Test
---------------------

a) Run /bin/x64/Debug/Odapter.exe and generate using _Odpt-SafeType.config.

b) Comment out all Define directives in Tester.cs except:

SAFETYPE_INTEGER          
SAFETYPE_NUMBER           
SAFETYPE_DATE             
SAFETYPE_TIMESTAMP        
ODPT_FILTER_PREFIX
MAPPING_FOR_TYPED_CURSOR
SEED_TABLES

c) Compile and run Tester project

b) Comment out all Define directives in Tester.cs except:
